# 快速开始

# 安装

在使用 Vue-CLI 3 或 4（推荐 4）创建的应用程序目录中，打开一个终端。

然后，通过运行以下命令，安装并调用 vue-cli-plugin-electron-builder 的生成器：

vue add electron-builder

好了！你已经准备好了！

# 启动开发服务器

如果你使用 Yarn (opens new window):

yarn electron:serve

或者，如果你使用 NPM:

npm run electron:serve

# 构建你的应用程序

使用 Yarn:

yarn electron:build

或使用 NPM:

npm run electron:build
